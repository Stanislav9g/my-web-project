🎯 Задачи исследования

  • Изучить теоретическую базу веб-инженерии.
  • Раскрыть понятия Frontend и Backend.
  • Исследовать причины отказа от Adobe Flash.
  • Создать обучающий веб-ресурс.

⚙️ Стек разработки

При создании сайта использовались:

  • HTML5: семантическая структура.
  • CSS3: современный дизайн и Flexbox.
  • JavaScript: интерактив и управление слайдами.
  • GitHub Pages: хостинг проекта.

⚠️ Полный отказ от Adobe Flash

Основные причины перехода на HTML5:

  • Низкий уровень безопасности (уязвимости).
  • Высокое потребление ресурсов процессора.
  • Отсутствие поддержки на мобильных устройствах.

О проекте

Цель работы: исследовать процесс эволюции веб-технологий для разработки обучающего веб-сайта.

Веб-технологии на примере ресторана

🏠

Frontend

Зал ресторана. Интерьер, меню и удобные кресла — то, что видит гость.

👨‍🍳

Backend

Кухня. Скрытая зона, где готовятся данные по строгим рецептам.

🤵

API

Официант. Передает ваш заказ на кухню и приносит готовый ответ.

Инструментарий

VS Code

Редактор кода для написания HTML, CSS и JS скриптов.

GitHub

Хостинг и контроль версий для публикации сайта в сети.

Заключение

В ходе выполнения проекта была изучена история развития веб-инженерии. Созданный сайт наглядно демонстрирует переход от закрытых плагинов к современным стандартам. Практическая значимость работы заключается в возможности использования данного ресурса как учебного пособия.
